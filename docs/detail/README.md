## <Tag>2018-08-31</Tag> [Web前端工作的尴尬何解？](./md/[2018-08-31]Web前端工作的尴尬何解？.md)
::: tip 摘要
前端工作面临着三个尴尬  

* 就是做个页面的，没有技术含量
* 工作量在各方的理解上不一，难以评估
* 接的后端的接口，实现了设计效果，传达产品意图，就是搬运工，自己干了什么反而说不清楚  

概括起来就是，要你干啥，你能干啥，要你有啥用
:::
